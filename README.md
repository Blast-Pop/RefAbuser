# RefAbuser

Advanced Python-based referral link testing tool using automated browser interaction.

---

## Disclaimer

This project is provided for educational and research purposes only.  
The author is not responsible for any misuse or illegal activities performed using this software.

---

## Goal

The purpose of this project is to demonstrate:

- Automation using Selenium and ChromeDriver
- Handling anti-bot protections (e.g. Cloudflare Turnstile)
- Testing referral link validity
- Proof-of-concept (PoC) for automation techniques

---

## Proof of Concept (PoC)

This tool demonstrates:

- Automated generation of referral codes
- Real browser interaction using `undetected-chromedriver`
- Basic human-like behavior simulation (delays, randomness)
- Detection of valid referral links via page content analysis
